mathom is a free, open-source disk space analyzer for Windows. It reads NTFS drives straight from the Master File Table — a couple of million files map in seconds, live, as a zoomable treemap.

Scripted demo with example data. The real app does exactly this to an actual 2,000,000-file drive — streaming in live at millions of files per second.

MFT-fast

On NTFS, mathom skips folder walking and reads the Master File Table directly — 2 million files in about 2.4 seconds on an ordinary NVMe laptop. No admin rights? It falls back to a fast parallel walker, automatically.

Live from the first batch

Results stream in while the scan runs — the tree and treemap grow in front of you instead of appearing after a progress bar. You're exploring before it finishes.

Honest sizes, safe deletes

Logical vs. allocated size, NTFS compression, sparse files, hardlinks counted once, OneDrive placeholders reported truthfully. Deletes go to the Recycle Bin, and Windows-critical folders are refused outright.

An alternative to WinDirStat, WizTree, and TreeSize

Coming from WinDirStat? Same job, same treemap idea — but scans finish in seconds instead of minutes, and the interface was designed this decade.

Coming from WizTree? mathom uses the same MFT-reading technique at comparable speed — faster on our test machine — and it's open source: MIT licensed, free for commercial use, no nag screens, and you can read every line of the scanner.

Evaluating TreeSize or missing DaisyDisk from your Mac? mathom is free for work machines by license, not by loophole — and it takes the visual side as seriously as the scan speed.
